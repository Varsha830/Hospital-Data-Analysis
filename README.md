# 🏥 Hospital Data Analysis Using SQL

## 📌 Project Overview

This project demonstrates how SQL can be used to analyze hospital operational data and generate meaningful business insights. The analysis focuses on patients, doctors, departments, hospital stays, medical expenses, and city-wise reports using MySQL.

The project answers common business questions by applying SQL aggregate functions, grouping, sorting, and date functions.

---

## 🎯 Objectives

- Calculate the total number of patients across hospitals.
- Determine the average number of doctors available in each hospital.
- Identify the top 3 departments with the highest patient count.
- Find the hospital with the maximum medical expenses.
- Calculate daily average medical expenses.
- Analyze the longest hospital stay.
- Generate city-wise patient reports.
- Calculate the average length of stay by department.
- Identify the department with the lowest patient count.
- Generate monthly medical expense reports.

## 📂 Dataset Information

The dataset contains hospital operational records with the following columns:

|-------------|-------------|

| Department | Hospital department |
| Doctors Count | Number of doctors |
| Patients Count | Number of patients |
| Admission Date | Patient admission date |
| Discharge Date | Patient discharge date |
